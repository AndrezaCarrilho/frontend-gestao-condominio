# Plataforma de Gestão Operacional - Frontend

Frontend da plataforma de gestão de condomínio, com base no layout do Figma.

## Tecnologias
- React.js
- Axios
- React Router DOM

## Estrutura
- Login
- Dashboard
- Moradores
- Veículos
- Visitantes
- Encomendas
- Reservas
- Comunicados

## Como rodar
```bash
npm install
npm start
```
